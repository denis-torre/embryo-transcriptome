Isoform-resolved human embryo transcriptome

This website is intended as a resource accompanying the publication "Isoform-resolved transcriptome reveals thousands of novel genes in human preimplantation embryos" (Torre et al.), which is currently under revision.

The resource contains the following sections:

  • Data files, where users can download all processed data files containing coordinates, structure and predicted biological properties for each isoform in the embryo transcriptome.
  • Genome browser, which contains an interactive UCSC genome browser session containing the full isoform-resolved dataset, alongside processed human embryo ATAC-Seq (Liu et al.). and CUT&RUN (Xia et al.) data, and multiple publicly available tracks (e.g. evolutionary conservation, repetitive element location, ENCODE, GTEx, etc.)
  • Isoform explorer, which allows users to interactively explore isoform-level predictions of biological function for each gene, and view both isoform and gene expression across the samples in our dataset.

Data files

Below, users can access all processed long- and short-read RNA-Seq data generated for this study, including isoform-level predictions of biological properties.
Note: the full dataset will be made publicly available upon publication of the manuscript, in the meantime, all files are available but limited to the first 100 rows.
